Privacy Policy

NotePin - AI Web Clipper for Notion · Last updated: 2026-05-06

1. Overview

NotePin ("we", "the extension") is a Chrome extension that lets you save web content to your own Notion database with AI-generated summaries, tags and categories. We respect your privacy. This document explains what data is processed, how it is used, and what is not collected.

2. We Do NOT Collect Anything

NotePin has no backend server. We do not collect, store, or transmit any of your personal data, browsing history, or content to our servers. We don't have servers.

  • ❌ No analytics or telemetry
  • ❌ No tracking pixels or cookies
  • ❌ No user accounts or sign-ups
  • ❌ No data shared with third parties (other than the APIs you explicitly configure)

3. What Data Is Processed

3.1 API Keys

You provide your own DeepSeek API Key and Notion Integration Token in the settings page. These keys are stored locally only in your browser via chrome.storage.local. They never leave your device except to authenticate requests to the official DeepSeek and Notion APIs.

3.2 Web Page Content

When you click "Save", the extension reads the content of the current web page (or your selection) and sends it to:

  • DeepSeek API — for summarization and tagging. See DeepSeek's Privacy Policy.
  • Notion API — to save the result into your own database. See Notion's Privacy Policy.

The page content is sent directly from your browser to those APIs. We have no intermediary server.

3.3 Local History

The extension keeps the latest 100 saves locally in your browser storage for the "Recent saves" list. This data never leaves your device. You can clear it any time via the popup.

4. Permissions Explained

  • activeTab / scripting: read the active tab's content when you trigger a save
  • storage: store your settings and local history
  • contextMenus: provide right-click "Save to Notion" menu
  • notifications: show success/failure notifications
  • host_permissions for api.deepseek.com and api.notion.com: call the respective APIs

5. Data Retention & Deletion

All data is stored locally in your browser. Uninstalling the extension removes all settings and history immediately. Content already saved into your Notion database remains under your own Notion account.

6. Children's Privacy

The extension is not directed to anyone under 13.

7. Changes to This Policy

If we change this policy, we will update the "Last updated" date and announce significant changes on the extension's homepage.

隐私政策

智存 (NotePin) - AI 智能网页收藏 · 最后更新：2026-05-06

1. 概述

智存（"我们"、"本扩展"）是一款 Chrome 扩展程序，帮助你把网页内容一键保存到自己的 Notion 数据库，并由 AI 自动生成摘要、标签和分类。 我们尊重你的隐私。本文说明哪些数据会被处理、如何使用，以及不会被收集的内容。

2. 我们不收集任何信息

智存没有任何后端服务器。我们不收集、不存储、也不向任何服务器传输你的个人数据、浏览历史或网页内容。我们没有服务器。

  • ❌ 无统计分析、无埋点
  • ❌ 无追踪像素、无 cookie
  • ❌ 无账号系统、无注册
  • ❌ 不与任何第三方共享数据（除你主动配置的 API 外）

3. 会处理哪些数据

3.1 API 密钥

你需要自己在设置页填入 DeepSeek API Key 和 Notion Integration Token。这些密钥只存在你本机浏览器， 通过 chrome.storage.local 保存。它们除了用于向 DeepSeek 和 Notion 的官方 API 发起请求之外，不会离开你的设备。

3.2 网页内容

当你点击"保存"时，本扩展会读取当前页面内容（或你选中的部分），并发送到：

  • DeepSeek API — 用于生成摘要和标签。详见 DeepSeek 隐私政策。
  • Notion API — 用于把结果保存到你自己的数据库。详见 Notion 隐私政策。

网页内容是从你的浏览器直接发送到上述 API，中间没有任何我们的服务器。

3.3 本地历史

为了在弹窗里展示"最近收藏"，扩展会在本地浏览器存储里保留最近 100 条记录。这些数据不会离开你的设备。 你可以随时在弹窗里清空。

4. 权限说明

  • activeTab / scripting：当你触发保存时读取当前标签页内容
  • storage：存储你的设置和本地历史
  • contextMenus：提供右键 "保存到 Notion" 菜单
  • notifications：展示成功/失败通知
  • host_permissions 针对 api.deepseek.com 和 api.notion.com：调用对应的 API

5. 数据保留与删除

所有数据均保存在你本机浏览器中。卸载本扩展会立即清除全部设置和本地历史。 已保存到 Notion 数据库中的内容由你的 Notion 账号自行管理。

6. 儿童隐私

本扩展不面向未满 13 岁的用户。

7. 政策变更

如有政策变更，我们会更新"最后更新"日期，并在扩展主页公告重要变化。
